# sreebhaskaran.github.io

Minimal academic website (single-page) inspired by lakshminittala.github.io.

## Files
- index.html (home)
- blog.html (blog index page)
- assets/style.css
- assets/script.js
- assets/CV_SreeBhaskaran.pdf (replace with your latest CV)
- assets/Sree-4-edit.jpg (your photo; upload this filename exactly)

## Deploy
Upload these files to the root of your GitHub Pages repo and enable Pages from main / root.
