# Personal academic website (GitHub Pages)

This is a simple single-page site (HTML/CSS/JS) for GitHub Pages.

## Deploy
1. Create a repo named **sreebhaskaran.github.io**
2. Upload `index.html` and the `assets/` folder to the repo root
3. Settings → Pages → Deploy from a branch → Branch: main → Folder: /(root)
4. Live at: https://sreebhaskaran.github.io/

## Customize
- `assets/style.css` controls colors (white background + SMU red/blue accents)
- Add Scholar/LinkedIn URLs near the top of `index.html`
- Replace the photo placeholder by swapping the `.avatar` div for an `<img>`
